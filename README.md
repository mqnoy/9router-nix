## Quick Start

```bash
# Clone and build
git clone https://github.com/mqnoy/9router-nix.git
cd 9router-nix
nix-build

# Run directly from the build output
./result/bin/9router
```

## Installation

### NixOS (system-wide)

Add the package to `environment.systemPackages` in your NixOS configuration:

```nix
# configuration.nix
{ pkgs, ... }:

let
  router9 = pkgs.callPackage /path/to/9router-nix/9router.nix {};
in
{
  environment.systemPackages = [ router9 ];
}
```

Then rebuild:

```bash
sudo nixos-rebuild switch
```

### Home Manager (standalone)

```nix
# ~/.config/home-manager/home.nix
{ pkgs, ... }:

let
  router9 = pkgs.callPackage /path/to/9router-nix/9router.nix {};
in
{
  home.packages = [ router9 ];
}
```

Then apply:

```bash
home-manager switch
```

### Home Manager (as NixOS module)

```nix
# configuration.nix
{ pkgs, ... }:

let
  router9 = pkgs.callPackage /path/to/9router-nix/9router.nix {};
in
{
  home-manager.users.mqnoy = {
    home.packages = [ router9 ];
  };
}
```

Then rebuild:

```bash
sudo nixos-rebuild switch
```

### Home Manager with Flakes

```nix
# flake.nix
{
  inputs = {
    nixpkgs.url = "github:NixOS/nixpkgs/nixos-unstable";
    home-manager.url = "github:nix-community/home-manager";
    router9-nix = {
      url = "github:mqnoy/9router-nix";
      flake = false;
    };
  };

  outputs = { nixpkgs, home-manager, router9-nix, ... }:
  let
    system = "x86_64-linux";
    pkgs = nixpkgs.legacyPackages.${system};
  in {
    homeConfigurations.mqnoy = home-manager.lib.homeManagerConfiguration {
      inherit pkgs;
      modules = [
        ({ pkgs, ... }: {
          home.username = "mqnoy";
          home.homeDirectory = "/home/mqnoy";
          home.stateVersion = "24.11";

          home.packages = [
            (pkgs.callPackage "${router9-nix}/9router.nix" {})
          ];
        })
      ];
    };
  };
}
```

Then apply:

```bash
home-manager switch --flake .#mqnoy
```

### NixOS with Flakes

```nix
# flake.nix
{
  inputs = {
    nixpkgs.url = "github:NixOS/nixpkgs/nixos-unstable";
    router9-nix = {
      url = "github:mqnoy/9router-nix";
      flake = false;
    };
  };

  outputs = { self, nixpkgs, router9-nix, ... }: {
    nixosConfigurations.<your-hostname> = nixpkgs.lib.nixosSystem {
      system = "x86_64-linux";
      modules = [
        ({ pkgs, ... }: {
          environment.systemPackages = [
            (pkgs.callPackage "${router9-nix}/9router.nix" {})
          ];
        })
      ];
    };
  };
}
```

Then rebuild:

```bash
sudo nixos-rebuild switch --flake .#<your-hostname>
```

```bash
./result/bin/9router --help
```

## Updating to a New Version

An automated `update.sh` script handles the entire update workflow.

### Auto-update to the latest npm version

```bash
./update.sh
```

This will automatically:

1. Query the npm registry for the latest `9router` version
2. Generate a new vendored `package-lock.json`
3. Compute the source tarball SRI hash
4. Discover the correct `npmDepsHash` via a trial build
5. Patch `version`, `hash`, and `npmDepsHash` in `9router.nix`
6. Run a final verification build

### Update to a specific version

```bash
./update.sh 0.4.66
```

### Check for updates without making changes

```bash
./update.sh --check
```

### After updating

Commit and push the changes:

```bash
git add 9router.nix packages/9router/package-lock.json
git commit -m "9router: <old-version> → <new-version>"
git push
```
